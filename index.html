playground

  • challenge: book finder (beginner)

    Create an application that will allow users to search for books by entering a query (Title, Author, etc). Display the resulting books in a list on the page with all the corresponding data.

  • challenge: christmas lights

    The ChristmasLights application relies on your development talents to create a mesmerizing light display. Your task is to draw seven colored circles in a row and based on a timer change the intensity of each each circle. When a circle is brightened it's predecessor returns to its normal intensity.

  • tech test: weather api work

    Create a project that consume the **geolocation endpoint**. Use the data from the geolocation endpoint to make a second call to the **forecast endpoint**. Use the data from both endpoints to generate the given creative.

  • tech test (old): responsive page based on desgin

    Create a responsive page according to the given creative.

  • tech test (old): accordion

    Create an accordion with the given data. Only 1 accordion should open at a time and the first should be open on page load. Use meaningful animations/transitions.
